# «Теплее!» — пакет документации

Telegram-игра: ежедневная семантическая головоломка (стиль Contexto/Semantle) + виральная механика «Загадай слово другу». RU-интерфейс, слова RU и EN.

## Состав пакета

| Файл | Что внутри |
|---|---|
| `CLAUDE.md` | Контекст для Claude Code. Положить в корень репозитория. |
| `docs/PRD.md` | Продукт: механики, UX, сценарии, метрики. |
| `docs/TECH_SPEC.md` | Архитектура, стек, схема БД, API, логика бота и Mini App. |
| `docs/DATA_PIPELINE.md` | Подготовка словарей и генерация рангов (офлайн, локальный ПК). |
| `docs/ROADMAP.md` | 7 этапов с критериями приёмки + готовые промпты для Claude Code. |
| `docs/DEPLOY.md` | Docker, GitHub Actions → SSH deploy, хостинг Mini App, HTTPS. |

## Порядок работы

1. Создай пустой репозиторий, скопируй туда `CLAUDE.md` и папку `docs/`.
2. Пройди `docs/DATA_PIPELINE.md` (Этап 0–1 роадмапа) — это делается локально, руками + скриптами. **Не пропускай ручную валидацию словаря.**
3. Дальше веди Claude Code по этапам из `docs/ROADMAP.md` — по одному этапу за сессию, с проверкой критериев приёмки.
4. Деплой по `docs/DEPLOY.md` (этап 6).

## Ключевые решения (зафиксировано)

- **Название:** Теплее! (username бота подобрать при регистрации).
- **Аудитория:** RU. Интерфейс на русском. Два режима слов: 🇷🇺 RU и 🇬🇧 EN (переключаются).
- **v1:** Daily-режим + челлендж «Загадай слово другу» (виральная петля).
- **v2 (потом):** дуэли на daily-слово, лидерборд друзей. **v3:** ко-оп в группах.
- **Сервер:** VPS 1GB RAM / 1 CPU, Docker. Никакого ML на сервере — только SQLite + матрица векторов guess-словаря (numpy, ~25–50 MB на язык).
- **Бот:** aiogram 3, **long polling** (webhook не нужен → бот работает без домена).
- **Mini App:** React + Vite, хостинг Cloudflare Pages (бесплатный HTTPS-домен `*.pages.dev`).
- **API:** FastAPI на VPS. ⚠️ Mini App может ходить только на HTTPS → API нужен домен с сертификатом. Решение: дешёвый домен (~$2–10/год) + Caddy (авто-HTTPS) **или** бесплатный Cloudflare Tunnel. Детали в `DEPLOY.md`.
- **CI/CD:** push в `main` → GitHub Actions → сборка образа в GHCR → SSH на VPS → `docker compose pull && up -d`. Mini App деплоится в Cloudflare Pages из того же workflow.
- **Смена дня:** 00:00 МСК, единый `day_id` для всех (без пользовательских таймзон в v1).
